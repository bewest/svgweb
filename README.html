The com/sgweb/svg directory contains the actionscript 3.0 source code for svgviewer.swf. svgviewer.swf is a flash "movie" used to support SVG (scalable vector graphics) files. See the FAQ file docs/FAQ.html for more information. The latest checked in binary is version 0.4. To see a sample SVG file, change directory to com/sgweb/svg/release and load one of the html files in your browser.
